## Use Cases

| Use Case | Goal | Outcome |
|----------|------|--------|
| Telemetry ingestion | Ingest telemetry from an industrial edge device into AWS | Data is received, validated, and stored |
| Data storage | Store telemetry in a durable and scalable format | Data is stored in S3 using structured partitioning |
| Observability | Provide visibility into ingestion pipeline | Logs and errors available via CloudWatch |
